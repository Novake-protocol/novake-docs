# Bônus Constelação

O Bônus Constelação são os pontos que acumulas a partir das Expedições dos membros da tua rede.

**Regra universal:** 1 ponto = $1

## Camada 1 — Expedição Tripulante

Quando qualquer membro da tua Constelação lança uma Expedição Tripulante, **toda a linhagem acima** recebe pontos.

- **Distribuição:** infinita em profundidade
- **Requisito:** nenhum — qualquer posição acima recebe
- **Valor dos pontos:** calculado com base nos Spoils da expedição

**Exemplo:**
- Membro da tua Constelação lança Expedição ASTRON $700, 90d
- Os Spoils presumidos geram pontos para toda a linhagem acima
- Tu e todos acima recebem os mesmos pontos

## Camada 2 — Expedição Comandante

Quando um membro lança uma Expedição Comandante, os pontos sobem pela linhagem mas com **requisito de activação**:

- **Distribuição:** infinita em profundidade
- **Requisito:** ter Nave + Expedição Comandante **activa**. Quem não tem → é PASSADO
- **Quantidade:** baseada na tua Patente

| Tua Patente | Capacidade de captura |
|---|---|
| Cadete | Captura base |
| Sargento | Captura intermédia |
| Comandante | Captura avançada |
| Supremo | Captura máxima |

**Pontos passados:** se não tens o requisito activo, os pontos passam para o próximo na linhagem que tem — são **irrecuperáveis**.

## Conversão em USDC

```
Bônus USDC = Total de Pontos × Factor do teu Pin
```

| Pin | Factor |
|-----|--------|
| Esmeralda | Factor base |
| Safira | Factor intermédio |
| Rubi | Factor avançado |
| Diamante | Factor superior |
| Black Diamond | Factor máximo |

O restante dos pontos vai para o treasury do protocolo — sustentando o ecossistema.

**Exemplo:**
- Acumulaste 5.000 pontos
- Tens Pin Black Diamond (factor máximo)
- Recebes a parte máxima em USDC
- O restante fortalece o treasury do protocolo

## Ver o Extrato

Na sidebar → **Spoils** → **Bônus Constelação** (`/bonus/contelacao`)

O extrato mostra:
- **Secção Comandante:** Membro | Nave | Valor | Pts capturados ✅ | Pts passados 💸
- **Secção Tripulante:** Membro | Nave | Plano | Pts recebidos ✅
