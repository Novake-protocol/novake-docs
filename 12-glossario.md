# Glossário NovaK Protocol

| Termo NovaK | Significado |
|---|---|
| **Spoils** | Ganhos gerados pelas Expedições Tripulante |
| **Nave** | NFT ERC-721 que representa a tua participação no protocolo |
| **Hangar** | Página de embarque em Naves |
| **Expedição** | Missão que a Nave realiza para gerar Spoils ou activar bônus |
| **Docking** | Estado da Nave disponível (DOCKED = pronta para Expedição) |
| **Plano de Voo** | Duração da Expedição (30/60/90/120 dias) |
| **Boarding Pass** | O teu link de convite único |
| **Constelação** | A tua rede de membros na estrutura de dois lados |
| **Esquadrão** | O lado escolhido pelo membro (Hyper Agents ou Hybrid Humans) |
| **Padrinho** | O membro que te convidou para o protocolo |
| **Comandante Supremo** | Membro de alta performance na Constelação |
| **Command Center** | O teu painel principal (Dashboard) |
| **War Room** | Sala de decisões estratégicas do protocolo |
| **Bônus de Embarque** | Bônus creditado ao padrinho quando membro embarca numa Nave |
| **Bônus Constelação** | Pontos gerados pelas Expedições dos membros da tua rede |
| **Bônus de Rede** | Bônus passivo activado pelos Spoils dos membros directos |
| **Patente** | Ranking: Cadete → Sargento → Comandante → Supremo |
| **Pin** | Conquista do Plano de Carreira: Esmeralda → Black Diamond |
| **Teto** | Cap máximo de bônus numa Expedição Comandante |
| **Spillover** | Membro que vai para posição mais abaixo (slots ocupados) |
| **Relançamento** | Reinvestir capital de Expedição encerrada numa nova Nave |
| **Nave Presente** | Nave oferecida pelo protocolo, com cap de colecta |
| **Janela de Colecta** | Período para colectar os Spoils |
| **Retorno ao Porto** | Colecta do capital principal |
| **$NVKE** | Token do protocolo (20M supply, Base L2) |
| **USDC** | Stablecoin usado nos embarques (Base mainnet) |
| **Base L2** | Blockchain onde o NovaK opera (Ethereum L2 da Coinbase) |
| **NFT** | Token único que representa cada Nave na blockchain |
