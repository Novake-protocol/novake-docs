# Glossário NovaK Protocol

| Termo NovaK | Significado |
|---|---|
| **Spoils** | Rendimentos/yield gerados pelas Expedições |
| **Nave** | NFT ERC-721 que representa o capital investido |
| **Hangar** | Página de compra de Naves |
| **Expedição** | Missão que a Nave realiza para gerar Spoils ou bônus |
| **Docking** | Estado da Nave em espera (DOCKED = pronta para Expedição) |
| **Plano de Voo** | Duração da Expedição (30/60/90/120 dias) |
| **Boarding Pass** | Teu link de convite único (código de referral) |
| **Constelação** | A tua rede de recrutas na árvore binária |
| **Comandante Supremo** | Líder de rede de alta performance |
| **Command Center** | O teu painel principal (Dashboard) |
| **War Room** | Sala de decisões estratégicas do protocolo |
| **Bônus de Embarque** | Comissão instantânea quando o teu recruta compra Nave |
| **Bônus Constelação** | Pontos binários de Expedições dos teus recrutas |
| **Bônus Residual** | % passiva do yield das Expedições Tripulante dos recrutas |
| **Patente** | Ranking: Cadete → Sargento → Comandante → Supremo |
| **Pin** | Conquista do Plano de Carreira: Esmeralda → Black Diamond |
| **Teto** | Cap máximo de bônus numa Expedição Comandante |
| **Spillover** | Recruta que vai para posição abaixo na árvore (slots ocupados) |
| **Redeploy** | Reinvestir capital de Expedição encerrada em nova Nave |
| **Nave Presente** | Nave GIFTED — dada pelo admin, sem capital real, cap de saque |
| **Janela de Coleta** | Período para sacar os Spoils (7d/48h) |
| **Retorno ao Porto** | Saque do capital principal (no aniversário do plano) |
| **$NVKE** | Token do protocolo (20M supply, Base L2) |
| **USDC** | Stablecoin usado nos pagamentos (Base mainnet) |
| **Base L2** | Blockchain onde o NovaK opera (Ethereum L2 da Coinbase) |
| **ERC-721** | Padrão de NFT único (cada Nave é um token distinto) |
| **ERC-20** | Padrão de token fungível ($NVKE) |
